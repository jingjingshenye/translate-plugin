## Quick Translate v1.0.0

划词翻译 + 弹窗翻译 Chrome 浏览器插件。

### 功能
- 18种翻译源（免费: Microsoft/Google/Tencent/Volcengine/Baidu/DeepL; AI: DeepSeek/OpenAI/Gemini/Claude等）
- 内置ECDICT本地词典（15000词 + 18789词形映射，支持词形还原）
- 划词翻译 + 弹窗翻译 + 在线词典 + 生词本

### 安装
1. 下载 `quick-translate-v1.0.0.tar.gz` 解压
2. Chrome 打开 `chrome://extensions/`
3. 开启「开发者模式」→「加载已解压的扩展程序」→ 选择 `extension` 文件夹
